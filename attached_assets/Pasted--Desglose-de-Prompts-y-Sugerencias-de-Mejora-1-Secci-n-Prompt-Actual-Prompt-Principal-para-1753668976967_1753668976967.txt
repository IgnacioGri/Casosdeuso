### Desglose de Prompts y Sugerencias de Mejora

1. **Sección/Prompt Actual**: Prompt Principal para Campos Regulares (clientName, description, etc.) - Incluye contexto (cliente/proyecto), tarea de mejora, reglas por campo, y devuelve solo valor mejorado.  
   **Alineación con Minuta ING**: Buena base: Cubre reglas como verbo infinitivo en useCaseName, formato XX000 en useCaseCode, y descripciones detalladas (alcance/objetivo). Minuta: Enfatiza secuencialidad inicial (pedir datos uno por uno), y descripciones con explicación de alcance/flujos. No menciona colores/font, pero prepara para export (Segoe UI Semilight, azul RGB para títulos).  
   **Sugerencias de Mejora**: - Mejora Prompt: Agrega placeholders para tipo de CU (entidad/API/proceso) de la minuta, ej. "Adapta a tipo [tipo]: para entidades incluye filtros/columnas; para APIs request/response". - Ejemplo Modificado: Añade "Asegura cumplimiento ING: descripciones 50-200 palabras con alcance detallado, flujos en listas indentadas si aplica." - Beneficio: Outputs más estructurados, listos para generación DOCX sin edición extra.

2. **Sección/Prompt Actual**: Reglas Específicas por Campo - Detalladas por campo (ej. clientName: empresa real; useCaseName: verbo infinitivo; description: 50-200 palabras técnicas).  
   **Alineación con Minuta ING**: Excelente para básicos: Verbo infinitivo obligatorio, formatos sin espacios. Minuta: Nombre CU en mayúsculas para título, archivo como "dos letras + 3 números + nombre CU". Descripciones deben incluir flujo principal (buscar/agregar para entidades).  
   **Sugerencias de Mejora**: - Mejora Reglas: Para useCaseName: "Comienza con verbo infinitivo (ej. Gestionar Entidades); prepara para título en mayúsculas RGB(0,112,192)". - Para description: "Incluye alcance con flujos principales (lista indentada: 1. Buscar [detalles filtros], a. Detallar columnas)." - Para fileName: "Exacto: [letras][números]_[NombreCU sin espacios], solicita letras/números si vacío." - Beneficio: Integra reglas de minuta directamente, evitando inconsistencias en export.

3. **Sección/Prompt Actual**: Prompt para Campos de Entidad (fieldsFromText) - Convierte descripción a JSON con name/type/mandatory/length.  
   **Alineación con Minuta ING**: Buena para estructuración: Tipos como text/number, mandatory basado en "obligatorio". Minuta: Para entidades, detallar cada dato con tipo/longitud/obligatorio/requerimientos especiales; agregar fecha/usuario alta/modificación automáticamente.  
   **Sugerencias de Mejora**: - Mejora Prompt: "Incluye auto: fechaAlta (date, mandatory), usuarioAlta (text, mandatory), fechaMod (date), usuarioMod (text). Output JSON con requerimientos especiales si menciona validaciones." - Ejemplo Modificado: "Reglas ING: Tipos: text/email/number/date/boolean; length si especifica; mandatory true para ID/fecha alta." - Beneficio: Asegura inclusión de campos obligatorios como en flujos de minuta (alta/modificación).

4. **Sección/Prompt Actual**: Campos con Procesamiento Inteligente Local (filtersDescription, columnsDescription, etc.) - Limpieza local, extracción patrones, numeración, adiciones automáticas (ej. "tiempo" → límite máximo).  
   **Alineación con Minuta ING**: Inteligente y eficiente (sin API calls extras). Cubre listas como filtros/columnas con formateo. Minuta: Filtros/columnas en listas indentadas para buscador; paginado/botones en bocetos; numeración multi-nivel (1-a-i con indent 0.2). Para wireframes: Boceto gráfico simplificado con paginado/botones Buscar/Limpiar/Agregar.  
   **Sugerencias de Mejora**: - Mejora Procesamiento: Para filters/columns: "Formatea como lista multi-nivel ING: 1. Filtro por [campo], a. Lógica [igual/mayor]. Indent 0.2 por nivel." - Para wireframeDescription: "Añade ING: Para buscador, incluye paginado, botones Buscar/Limpiar/Agregar/Editar/Eliminar; describe UI textual (ej. 'Tabla con columnas: [lista]')." - Para alternativeFlow/businessRules: "Numeración 1-a-i; para alternos incluye modificar/eliminar con verificaciones relaciones; agrega pre/postcondiciones si detecta." - Para specialRequirements: "Incluye configurables de minuta (ej. path/URL para procesos, usuario/clave)." - Beneficio: Outputs listos para HTML/DOCX, con indentación preservada.

5. **Sección/Prompt Actual**: Funciones de Procesamiento de Texto (cleanInputText, formatProfessionalText) - Limpieza de comillas/espacios, capitalización, punto final.  
   **Alineación con Minuta ING**: Simple y útil para consistencia profesional. Minuta: Todo en Segoe UI Semilight, interlineado simple; títulos azul RGB; tablas con anchos/bordes específicos (ej. historia revisiones).  
   **Sugerencias de Mejora**: - Mejora Funciones: Agrega "Prepara para export ING: Asegura texto plano listo para <strong> negrita en títulos, listas con indent." - Beneficio: Facilita parsing en docx.js para preservar formatos (ej. negrita en headers de tablas).
